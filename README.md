# Lumina Odontologia

Landing page estática para uma clínica odontológica premium fictícia, pensada como peça de portfólio front-end.

## Visão geral

Este projeto apresenta uma landing page comercial com foco em:

- autoridade visual para serviços de saúde
- hierarquia clara de conversão
- narrativa de confiança, tecnologia e acolhimento
- responsividade e microinterações leves

## Posicionamento do case

A `Lumina Odontologia` é uma marca conceitual criada para demonstrar direção visual, estrutura de landing page e capacidade de acabamento front-end sem depender de bibliotecas ou frameworks.

O formulário presente na página é um formulário demonstrativo: ele existe para mostrar como um bloco de captação pode ser apresentado em um projeto comercial e pode ser conectado depois a WhatsApp, e-mail, CRM ou automações.

## Stack

- HTML5
- CSS3
- JavaScript nativo

## Arquivos principais

- `index.html`: estrutura e conteúdo da landing
- `style.css`: layout, tema visual, responsividade e animações
- `script.js`: menu mobile, reveals on scroll e comportamento demonstrativo do formulário
- `favicon.svg`: ícone usado na aba do navegador e nos metadados básicos da página

## Como visualizar

Abra `index.html` no navegador ou publique o repositório em GitHub Pages.

## Destaques do refinamento

- remoção de placeholders evidentes
- links coerentes para os CTAs principais
- metadados básicos para compartilhamento
- favicon próprio
- imagens secundárias com `loading="lazy"`
- mensagem honesta para o bloco de conversão

## Observações

- a marca é conceitual, criada apenas para o case
- o formulário é demonstrativo e pode ser integrado depois a WhatsApp, e-mail, CRM ou automações
